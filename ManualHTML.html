-Como empezar un documento HTML

	Para empezar un documento HTML lo primero que debemos saber es que cada elemento que pongamos en el texto va en una etiqueta. Las etiquetas se 
ponen con los simbolos de mayor y menor que. En la primera etiqueta se debe poner "!doctype html" y cerrar la etiqueta para saber que es un documento
html. Luego debemos dividir el documento abriendo la etiqueta "html", dentro de esta deben ir 2 etiquetas, la etiqueta "head", y una vez se cierra
esta, la etiqueta "body".
	La etiqueta head sirve para poner datos del documento, pero que no seran visibles en el, usando las etiquetas "meta", por ejemplo podemos poner
el nombre del autor de la pagina abriendo la etiqueta meta y poniendo meta name="author" content="nombre de autor" y cerando la etiqueta. Cuando se pone
una palabra seguido del simbolo igual y despues otra palabra entre comillas indica un atributo. En este caso "author" quiere decir el nombre del autor
del texto.
	En la etiqueta body ponemos lo que quedara visible en el documento. Se pueden poner imágenes, tablas, o simplemente textos en los que habra cambios
como por ejemplo texto en negrita, parrafos, hiperenlaces... Siempre debemos abrir y cerrar las etiquetas. Las etiquetas se cierran poniendo la barra "/" 
dentro de la etiqueta, por ejemplo, para abrir body seria "body" entre simbolos de menor y mayor, y para cerrarlo "/body" entre los simbolos de mayor y
menor.

-Como crear titulos

-Como crear listas

	Para crear una lsita debemos saber que hay dos tipos de listas. Las listas ordenadas y las listas desordenadas.
	Las listas ordenadas se separaran generalmente por números, pero añadiendoles un atributo podremos cambiarlo.
	Las listas desordenadas se separan por puntos, pero tambien podremos cambiarlo con otro atributo.
	Para empezar una lista debemos poner la etiqueta "ol" para las listas ordenadas, y "ul" para las listas desordenadas. Una vez abierta la etiqueta
en cada elemento de la lista se debe poner la etiqueta "li". Esta etiqueta no se debe cerrar. Una vez hemos puesto los elementos de lista deseados volvemos
a cerrar la etiqueta "ol" para las ordenadas, o "ul" para las desordenadas. Vamos a ver un ejemplo de ambas.

-Como resaltar el texto

	Para resaltar una palabra o texto en negrita, italica (o bastardilla) o subrayada, debemos usar las etiquetas "i", "b" y "u" respectivamente.
Debemos abrirlas donde queramos modificar el texto o palabra, y cerrarla para dejar de hacerlo.
	En esta linea vamos a mostrar un ejemplo de negrita, otro de italica, y por ultimo un ejemplo de subrayada.
	Tambien podemos mostrar algo irrelevante con la etiqueta "s", abriendo y cerrandola donde queramos que el texto este tachado.
Podemos recalcar algo del texto con la etiqueta "em", se parece a la italica, pero es distinta. Podemos mostrar algo de texto en letra
grande para señalar algo importante, serio o urgente, con la etiqueta "strong", y por ultimo mostrar algo en pequeño
con la etiqueta "small". Esto sirve para representar por ejemplo la letra pequeña de un contrato. La etiqueta small se diferencia poco, pero para 
mostrarlo mejor voy a usar dos ejemplos.

-Como hacer hiperenlaces

	Como habras visto ya se pueden hacer hiperenlaces. Podemos hacer elaces una pagina en internet, un sitio del documento e incluso a otro
documento externo.
	Para hacer un enlace a pagina de internet debemos usar la etiqueta "a" con el atributo "href="pagina de internet". Por ejemplo vamos a
hacer un enlace a la pagina web del instituto. Abrimos la etiquta "a" con el atributo "href" escribimos una palabra o texto donde estara el
enlace, y cerramos la etiqueta "a", por ejemplo:  Ir a la pagina del instituto
	Si quisieramos hacer un enlace a otro documento externo deberiamos usar la etiqueta "a" con el atributo "name="direccion documento externo".
	Por ultimo para hacer un enlace a una parte del documento en el que estamo debemos crear un elemento en el lugar dodne queramos ir
poniendo la etiqueta "elto id=nombre elemento" en un lugar del texto. Luego debemos usar "a href=#nombre elemento" y escribir algo, como por ejemplo
Click aqui para ir al indice. Recuerda siempre cerrar la etiqueta "a" y la etiqueta "elto". 
	También podemos hacer enlaces clickando, por ejemplo, en una imagen. Lo veremos a continuacion en el apartado de añadir imagenes.

-Como introducir imagenes


	Para introducir una imagen debemos usar la etiqueta "img". Si no añadimos un atributo de altura o anchura, la imagen cogera el tamaño
 por defecto, pero usando el atributo "width" y "height" podemos modificar el ancho y alto de la imagen. Si solo ponemos 1 de ellos la imagen
se ensanchara en ambos lados para no deformar la imagen. Podemos poner una imagen que tengamos descargada o otra imagen de internet poniendo
la url. vamos a poner de ejmplo una imagen de internet, pero vamos a modificar el tamaño ya que es demasiado grande.

	La imagen se ha modificado solo en la altura, pero ha modificado el tamaño proporcionalmente en la anchrua tambien. Ahora vamos a poner
otra imagen y modificar ambas cosas para ver como se deforma.

	Por ultimo podemos hacer que clicando en un sitio especifico de una imagen nos lleve a un sitio de internet o un documento, es decir, 
podemos hacer un hiperenlace. Para hacer el hiperenlace debemos usar la etiqueta "img" con el atributo "usemap=#nombre del mapa". Una vez puesta
la etiqueta "img" debemos poner una etiqueta "map name=#nombre del mapa" y despues de esta otra etiqueta llamada "area" donde seleccionaremos si
queremos hacer un cuadrado, rectangulo, circulo o cualquier otra figura. Despues daremos las cordenadas separadas por comas, la direccion a donde
nos llevara al hacer click y el texto que pondra con el raton encima si es un hiperenlace. Son atributos que se ponen en "area". Vamos a hacer un

-Como hacer tablas


	Por ultimo vamos a ver como hacer tablas en html. Como sabras las tablas se componen de filas (de izquierda a derecha), y de columnas 
(de arriba a abajo). Debemos usar la etiqueta "table" para iniciar una tabla, la etiqueta "tr" para iniciar una fila, y la etiqueta "td" para
iniciar una columna. Tambien podemos ponerle un titulo a la tabla con la etiqueta "th". La unica etiqueta que debemos cerrar es la etiqueta
"table". Podemos ponerle atributos a la tabla, como por ejemplo "border" para que tenga un borde. y a las etiquetas "th" y "td" se les pone
los atributos "rowspan" para expandir en columna, y "colspan" para expandir en fila. Vamos a hacer una calculadora como ejemplo.
